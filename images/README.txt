Images Directory
================

Place your images in this directory:

1. profile.jpg - Your main profile photo for the homepage hero section
2. about-photo.jpg - A photo of yourself for the About Me page
3. project1.jpg - Screenshot or image for your first project
4. project2.jpg - Screenshot or image for your second project
5. project3.jpg - Screenshot or image for your third project (optional)

Image recommendations:
- Use high-quality images (at least 1000px wide)
- Profile photo should be square or portrait orientation
- Project images should be landscape orientation (16:9 ratio recommended)
- Optimize images for web (compress to reduce file size)
- Supported formats: JPG, PNG, WebP
